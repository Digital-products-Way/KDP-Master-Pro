# KDP Master Pro

Accurate cover size calculator for Amazon KDP paperbacks.

## Live Tool
https://digital-products-way.github.io/KDP-Master-Pro/

## Preview
![KDP Master Pro Screenshot](Screenshot_20260117_22194.png)

## What it does
- Calculates spine width using paper thickness
- Gives full cover size (front + back + spine)
- Supports bleed / no bleed
- Ready sizes for Canva & Photoshop
- Mobile-friendly, instant results

## Privacy
No login • No ads • No data collection
